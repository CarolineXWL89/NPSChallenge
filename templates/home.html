Welcome to the National Park Service Info Page

Overview

This site was created by Caroline Liu as part of the Capital One Tech Challenge, summer 2019. The challenge involves creating a web application that serves as an info kiosk for the 330 million visitors and 40 million overnight campers per year.
The challenge consists of four deliverables:
  1. Provides tools to assist users in finding specific information, such as state and designation filtering, name and keyword search.
  2. Lists details about specific visitor centers, as well as nearby campgrounds.
  3. Displays alerts, articles, and news releases about a selected destination.
  4. Provides educational information about a selected destination, utilising availble lesson plans as well as relevant people and places associated with the location.

There are additional features that were suggested, including the following:
  • When displaying information, utlise relevant symbols from the NPS Symbol Library to illustrate relevant items.
  • Utlise latitude and longitude data for map visualisation or data calculations.

Here is a quick run-down of the logistics of the site, both in the features as well as some potentially problematic areas:
  1. The home and about page are just for aesthetical purposes. The images used have been found on Google and have been credited where I could such that I'm not violating copyright policies.
  2. The slideshow at the top of the homepage has been formatted correctly and I've done everything I can to prevent it from jumping around but even so, I haven't been able to get to the very bottom of the issue.
  3. This is my very first attempt at web-development, so I attempted making a cache in order to make the API calls a tiny bit faster (they're a bit too slow to use normally at the moment) but I haven't figured out a way in order to do it without writing every possible call into a text file, hence it's last on my priority list.
